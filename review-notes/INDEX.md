## 高频考点速查

- **Cloud Models**: IaaS / PaaS / SaaS / FaaS 对比
- **Networking**: TCP vs UDP, VPC, Security Groups, CIDR
- **IAM**: Least Privilege, Roles vs Users, Policies
- **Storage**: S3 / EBS / EFS 对比, Object vs Block vs File
- **Scaling**: Horizontal vs Vertical, Auto Scaling, Load Balancer
- **DNS**: A/CNAME/MX 记录, Route 53, TTL
- **Observability**: Metrics / Logs / Traces 三支柱
- **Microservices vs Monolith**: 优劣势、拆分策略
- **Serverless**: Lambda 触发器、冷启动、适用场景
- **Security**: Encryption at rest/in transit, OWASP, Least Privilege
- **12-Factor App**: 12 个要素的含义与应用
- **IaC / Terraform**: state file, plan/apply, idempotency
